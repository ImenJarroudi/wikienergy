Utilities¶

Given a list of temporally aligned instances, aggregate them into a single signal.

disaggregator.utils.aggregate_traces(traces, metadata, how='strict')[source]¶

Given a list of temporally aligned traces, aggregate them into a single signal.

disaggregator.utils.concatenate_traces(traces, metadata=None, how='strict')[source]¶

Given a list of appliance traces, returns a single concatenated trace. With how=”strict” option, must be sampled at the same rate and consecutive, without overlapping datapoints.

disaggregator.utils.concatenate_traces_lists(traces, metadata=None, how='strict')[source]¶

Takes a list of lists of n traces and concatenates them into a single list of n traces.

disaggregator.utils.get_common_ids(id_lists)[source]¶

Returns a list of ids common to the supplied lists. (id set intersection)

disaggregator.utils.order_traces(traces)[source]¶

Given a set of traces, orders them chronologically and catches overlapping traces.

disaggregator.utils.pickle_object(obj, title)[source]¶
